# Visualizador 3D de Produto

Este é um visualizador 3D simples construído com Three.js para visualizar e personalizar modelos glTF.

## Como Executar

Por razões de segurança, os navegadores bloqueiam o carregamento de arquivos locais (`.glb`) se você abrir o `index.html` diretamente no navegador (Erro de CORS).

Para visualizar o projeto corretamente, você precisa de um servidor local:

### Opção 1: Usando NPM (Recomendado)

Se você tem o Node.js instalado:

1.  Abra o terminal na pasta do projeto.
2.  Execute o comando:
    ```bash
    npm start
    ```
3.  Acesse `http://localhost:3000` no seu navegador.

### Opção 2: VS Code Live Server

Se você usa o VS Code:

1.  Instale a extensão **Live Server**.
2.  Clique com o botão direito no `index.html` e selecione **Open with Live Server**.
